# 🎭 1-Min Actor

A simple daily acting practice tool designed to build confidence, expression, and camera comfort — one minute at a time.

---

## 🌟 Purpose

This app was built to solve a simple problem:

> “I want to practice acting, but I don’t know what to do daily.”

It removes overthinking and provides structured, repeatable practice.

---

## ⚙️ Features

### 🎲 Prompt Generator
- Situation + Emotion + Dialogue
- Optional emotion filtering
- Smart randomization (no repetition)

---

### 🎭 Emotion Practice
- One line → multiple emotions
- Select specific emotions to focus
- Helps build emotional range

---

### 🎬 Scene Practice
- Inspired by Bollywood movies
- Short, beginner-friendly scripts
- Focus on expression + delivery

---

### ⏱️ 1-Min Timer
- Encourages focused practice
- Builds consistency

---

### 🎥 Recording Feature
- Record yourself directly
- Playback for self-review
- Helps improve confidence

---

## 🧠 Philosophy

This is not an acting course.

This is a **confidence-building system**:
- Practice without pressure
- Improve through repetition
- Focus on expression, not perfection

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Local JSON (no backend)
- MediaRecorder API
